# LingoEcho - 极简语音背单词助手

LingoEcho 是一款专为高效记忆单词设计的极简 Web 应用。它结合了智谱 AI (GLM) 的强大能力与浏览器原生技术，提供清新的交互体验，帮助用户通过听觉与视觉的双重反馈来加固单词记忆。

## ✨ 功能特性

- **🌿 清新设计**：基于 Plus Jakarta Sans 字体与现代毛玻璃质感 UI，营造专注的学习氛围。
- **🤖 智谱 AI 驱动**：
  - **AI-TTS (云端语音)**：使用 `glm-tts` 引擎，提供极其自然的真人配音，解决移动端/微信无声问题。
  - **Vision OCR (视觉识词)**：使用 `glm-4.6v-flash` 模型，支持拍照或上传相册提取教材单词。
- **🎙️ 语音录入**：支持通过 Speech Recognition 快速录入单词，无需手动打字。
- **🧠 记忆模式**：
  - 单词顺序随机打乱。
  - 默认隐藏单词，增强主动回忆效果。
  - **智能引擎切换**：优先使用本地语音（省流量/低延迟），在受限环境（如微信）自动切换至 AI 云端语音。
- **🛠️ 系统诊断**：内置诊断中心，实时检测 API 连通性、播报状态及运行环境。

## 🔑 API Key 配置指南 (核心)

为了启用 **AI 语音播报** 和 **拍照识词** 功能，您需要配置智谱 AI 的 API Key：

### 1. 获取 API Key
1. 访问 [智谱 AI 开放平台 (bigmodel.cn)](https://bigmodel.cn/)。
2. 注册并登录账号。
3. 进入 **控制台** -> **API Key** 页面。
4. 复制您的专属 API Key。

### 2. 在 Vercel 部署时配置 (推荐)
如果您使用 Vercel 一键部署，请在项目设置中添加环境变量：
- **Variable Name**: `GLM_API_KEY`
- **Value**: `您的 API Key 内容`
- 添加后重新执行部署即可生效。

### 3. 本地开发配置
在项目根目录创建 `.env` 文件（或在 shell 中临时导出）：
```bash
# 示例
GLM_API_KEY=your_actual_api_key_here
```

## 🚀 部署步骤

### Vercel 部署
1. **关联仓库**：将代码推送至 GitHub。
2. **导入项目**：在 Vercel 中选择该仓库。
3. **配置变量**：在 "Environment Variables" 栏目添加上述的 `GLM_API_KEY`。
4. **点击 Deploy**：等待约 1 分钟即可在线访问。

## 📖 使用方法

### 1. 创建单词列表
- 点击右上角的 **“新建词单”**。
- **手动输入**：直接在文本框输入。
- **AI 提取**：点击“拍照识词”或“相册提取”，AI 将自动解析图片中的英文单词。
- **语音录入**：在非移动端环境下，点击“英文/中文语音”直接开口朗读录入。

### 2. 开始听写
- 点击词单卡片上的 **“开始听写”**。
- **TTS 引擎状态**：播报按钮上方会显示当前驱动引擎（Offline 代表本地，AI Cloud 代表智谱语音）。
- **重播与切换**：点击中心旋转图标重听，使用左右箭头切换进度。

### 3. 环境诊断
- 如果听不到声音，点击右上角的 **“小虫子”图标** 进入诊断中心。
- 在“API 连通性”测试您的 Key 是否有效。
- 在“语音实验室”对比本地与云端音质。
- 在“环境信息”查看当前设备是否受限。
